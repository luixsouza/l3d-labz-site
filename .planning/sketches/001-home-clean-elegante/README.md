# Sketch 001: Home — Clean & Elegante

## Design Question
O visual atual é dark quase-preto (`#0a0d15`) e parece "sombrio". Esta direção
light-first (off-white quente, verde Luigi contido + azul secundário, sombras
difusas) resolve isso **e** promove o visualizador 3D — o core value — pro hero?

## How to View
open .planning/sketches/001-home-clean-elegante/index.html

> Requer internet (fontes Google + script do `<model-viewer>` + GLB de exemplo via CDN).

## Variants
- **A — Antes (dark atual)** — reproduz o visual sombrio de hoje, só pra comparar lado a lado.
- **B — Clean, hero split ★** — proposta principal: texto/CTA à esquerda, `<model-viewer>` girando à direita, com botão "Baixar STL". Inclui chips de categoria coloridos.
- **C — Clean, 3D protagonista** — hero centralizado com o 3D grande e full-width; aposta mais agressiva no diferencial.

## What to Look For
- **Clima geral:** alterne A↔B. O quanto o off-white + sombras difusas tira o peso "sombrio".
- **Papel do 3D:** B coloca o viewer ao lado do discurso; C faz dele o herói. Qual vende melhor o diferencial?
- **Cor:** verde só em CTA/detalhe, azul em links/badges/prova social — equilíbrio "cores do Luigi" sem poluir.
- **Densidade:** respiro entre seções, cards com sombra (não borda dura), chips de categoria.
- Use a toolbar (canto inf. dir.) pra testar Phone/Tablet/Full.
